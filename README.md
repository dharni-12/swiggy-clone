#  Swiggy Clone - Online Food Ordering System

**Domain:** Web Development / Food Tech

##  Project Overview
A web-based food ordering interface inspired by Swiggy,
implementing restaurant listings, search/filter functionality,
and a fully working cart system.

---

## 🎨 Features
-  Live search by restaurant name or cuisine
-  Price range filter
-  Rating filter
-  Add to cart with quantity controls
-  Checkout with order confirmation
-  Fully responsive design

---

##  Technologies Used
- HTML5
- CSS3 
-  JavaScript
